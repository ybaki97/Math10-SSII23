Polynomial Regression#

We‚Äôve already seen examples of performing linear regression with multiple input features. If we can do this, then performing polynomial regression is also possible.

Key Point: Polynomial regression is no more difficult than linear regression.

Think back to the taxis dataset where we performed linear regression on multiple input columns: $\( \text{duration} = c_1*\text{passengers} + c_2*\text{hour} + c_3*\text{distance} + c_0 \)$

Imagine now we want to fit to a degree three polynomial:

\[ y \approx c_0 + c_1x^1 + c_2x^2 + c_3x^3 \]

Imagine now that the difference powers of \(x\) are all just different columns in the data that we‚Äôre fitting to.

Warnings about polynomial regression#

We might be pretty impressed by the fit above! It looks like the degree three polynomial fits the data well, but this is in part due to a problem called overfitting. In general, the better a fit appears to be, the worse it will perform on future data. Part of the issue with large degree polynomials is that we have a lot of flexibility/choice in the coefficients.
